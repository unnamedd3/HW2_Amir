# Краткое описание программы

Консольное приложение для управления рецептами.

Программа позволяет создавать рецепты, добавлять в них ингредиенты, пересчитывать
количество продуктов для другого числа порций и формировать общий список покупок.

## Установка

Склонируйте репозиторий и перейдите в папку проекта:

```bash
git clone <ссылка-на-репозиторий>
cd <папка-проекта>
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Структура проекта

HW2/
- main.py
- recipes.py
- test_recipes.py
- requirements.txt
- README.md
- .gitignore


- `main.py` — консольное приложение с меню.
- `recipes.py` — классы `Ingredient`, `Recipe`, `DietaryRecipe`, `ShoppingList`.
- `test_recipes.py` — тесты для проверки работы классов.
- `requirements.txt` — зависимости проекта.
- `.gitignore` — файлы и папки, которые не нужно добавлять в Git.

## Использование

Запуск приложения:

```bash
python3 main.py
```

В приложении можно:

- создать новый рецепт;
- добавить продукт в рецепт;
- посмотреть все рецепты;
- пересчитать рецепт по коэффициенту;
- добавить рецепт в список покупок;
- посмотреть список покупок.

## Тестирование

Запуск тестов:

```bash
python3 -m pytest
```

После проверки должно появиться сообщение, что все тесты пройдены.
